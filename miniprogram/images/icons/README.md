# Icons Required

Place PNG icons (81x81px recommended) in this folder:

## TabBar Icons (required)
- search.png / search-active.png
- history.png / history-active.png
- star.png / star-active.png
- tools.png / tools-active.png (元素周期表/化学工具)
- settings.png / settings-active.png

## Component Icons (required)
- search-gray.png (search input icon)
- close.png (close/clear button)
- filter.png (filter button)
- star-outline.png (unfavorited)
- star-filled.png (favorited)
- arrow-right.png (list arrow)
- share.png (share button)
- copy.png (copy button)
- pdf.png (PDF icon)
- export.png (export icon)
- delete.png (delete button)
- material.png (material type icon)
- literature.png (literature type icon)

## Empty State Icons
- empty.png
- search-empty.png
- history-empty.png
- star-empty.png
- error.png
- network-error.png
- search-big.png (initial search state)

Generate these icons using any icon library (e.g., Feather, Heroicons) or design tool.
